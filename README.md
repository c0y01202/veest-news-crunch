 ## Installation:

   To install dependancies, run the following commands:

     ```
     npm install:
    express
    node
    heroku
    sequelize
    mysql2
    express-handlebars
    init
    dotenv
    bcrypt
    express-session
    connect-session-sequelize
     ```
